A complete pipe inventory management system with dynamic forms and dashboard.

== Description ==

This plugin provides:
1. Custom pipe inventory form with dynamic fields
2. Dashboard for viewing submissions
3. Settings page for configuring agents, areas, and stores
4. Support for multiple pipe types with size/length dependencies

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-pipe-inventory` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Pipe Inventory > Settings to configure agents, areas, and stores
4. Use the [pipe_inventory_form] shortcode to display the form on any page

== Features ==

- Dynamic form fields with dependencies
- Multi-row container with add/remove functionality
- Dashboard for viewing submissions
- Configurable agents/areas/stores
- Support for multiple pipe types (SENBAGA, UPVC, YOGA, ISI, HDPE)
